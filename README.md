# game-flybird-pygame
"Powerpuff Girls Extreme Flight" is a classic side-scrolling flight game developed with Python + Pygame, replicating and optimizing the core gameplay of Flappy Bird.

## 项目概述
- **项目定位**：基于 Python + Pygame 开发的横版飞行穿越游戏（复刻并优化 Flappy Bird 核心玩法），聚焦游戏开发中的图形渲染、物理逻辑模拟与交互设计，是 Python 工程实践与游戏开发基础能力的具象化项目。
- **技术栈**：Python 3.8+, Pygame 2.1.0
- **开发周期**：2 周（独立开发，覆盖需求拆解、技术选型、编码实现、测试优化全流程）
- **核心目标**：通过轻量化游戏开发，落地 Python 面向对象编程、资源管理、碰撞检测、物理引擎模拟等核心工程技能。

## 核心功能
1. **完整游戏流程闭环**：菜单页（空格键触发启动）→ 游戏页（操控角色穿越水管）→ 结局页（20 分解锁通关/未达标显示失败），交互逻辑清晰且无断点。
2. **物理引擎模拟**：实现重力加速度（`gravity=1`）、飞行速度限制（`max_y_vel=10`）、角色角度动态旋转（初始 20° 至最大下落角度 -20°），还原真实飞行力学手感。
3. **随机关卡生成**：水管间距（220-250px）、纵向位置随机分配，生成间隔动态适配，提升游戏可玩性与抗重复性。
4. **多场景音频反馈**：背景音、拍打翅膀音、得分提示音、碰撞死亡音全流程覆盖，强化交互沉浸感。
5. **精准碰撞检测**：基于 Pygame Sprite 精灵组实现角色与水管、屏幕边界的碰撞判断，响应延迟＜10ms，无误判情况。
6. **实时得分可视化**：以水管中心为判定点，精准统计穿越次数，得分通过数字精灵实时渲染展示，视觉清晰。

## 技术实现亮点
### 1. 面向对象设计（OOP）
- 封装 `Bird` 类（管理飞行状态、速度、角度旋转）与 `Pipe` 类（处理水管生成、移动、复用），职责单一，代码复用率提升。
- 抽象图片/音频资源加载逻辑，通过字典统一管理，降低模块耦合，支持后续资源快速扩展。

### 2. 性能优化策略
- 水管对象复用机制：移出屏幕的水管实例直接销毁并重新生成，避免内存泄漏，确保游戏长时间运行无卡顿。
- 帧率稳定控制（`FPS=30`）：通过 `pygame.time.Clock()` 强制同步帧率，保证不同设备下运行效果一致。

### 3. 交互体验优化
- 空格键响应优先级设计：支持连续输入，飞行操控无延迟，手感流畅。
- 死亡动画与音效同步：角色碰撞后触发 90° 旋转+垂直下落动画，搭配专属音效，反馈直观。

## 快速运行
### 1. 环境准备
确保你的设备已安装 Python 3.8 及以上版本（推荐 3.8-3.11，兼容性最佳），无需额外配置复杂环境。
### 2. 安装依赖
打开终端 / 命令提示符，直接执行以下命令安装 pygame（无需额外文件，安装完成即可运行）：
pip install pygame==2.1.0  # 稳定兼容版本，适配项目所有功能
### 3. 启动游戏
下载项目源码压缩包，解压后找到 flybird.py 文件；
